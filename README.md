# 🟩 EmoHabit — The Matrix of Consistency

A deeply personalized, GitHub-style habit tracker designed to help you maintain consistency and momentum. Built with an oceanic bioluminescent glassmorphism UI.

## Features
- **Offline First**: All your habits are securely saved in your browser's local storage. No backend needed.
- **GitHub-Style Heatmap**: Visualizes your progress over the last 365 days. The longer your streak, the brighter the grid square glows!
- **Dynamic Streak Calculation**: Tracks your current streak, longest all-time streak, and total completions.
- **Bioluminescent Aesthetic**: A beautiful dark theme using `Outfit` and `Inter` typography.

## Getting Started
1. Clone this repository.
2. Open `index.html` in your browser.
3. Start adding habits!
